# shoto

A Claude Code marketplace hosting plugins for **building Claude Code itself**.

## Install

```bash
# inside Claude Code
/plugin marketplace add shoto290/shoto
/plugin install <plugin>@shoto
```

## Plugins

| Plugin | Description |
| :--- | :--- |
| [`evolve`](./plugins/evolve) | Meta-tooling to create and evolve Claude Code skills, sub-agents and hooks. |
