The `utils` plugin (v0.3.0) provides foundational skills that other shoto plugins depend on. It has no sub-agents — only skills you invoke directly with `/utils:<name>`.

## Overview

`utils` is the shared layer of the shoto marketplace. It handles two recurring tasks that every plugin eventually needs: knowing which Claude Code built-in tool to reach for, and bumping a plugin's semver version from its commit history.

Install it once and every plugin that lists `utils` as a dependency can call these skills without additional setup.

## Install

```bash
/plugin install utils@shoto
```

If you haven't added the shoto marketplace yet, run this first:

```bash
/plugin marketplace add shoto290/shoto
```
