The `frontend` plugin (v0.1.0) is a knowledge base of battle-tested frontend patterns. It has a single auto-invoked skill that applies when you are building or auditing fast web applications.

## Overview

`frontend` captures production-grade patterns distilled from real architectures — specifically Linear's. Rather than generic advice, it gives you concrete rules for the data layer, initial load, runtime cost, and input model of a web app.

The skill is auto-invoked: Claude loads it automatically when your conversation involves topics like bundle size, code splitting, IndexedDB, optimistic UI, animations, or keyboard shortcuts.

**Version:** `0.1.0`

## Install

```bash
/plugin install frontend@shoto
```

If you haven't added the shoto marketplace yet, run this first:

```bash
/plugin marketplace add shoto290/shoto
```
