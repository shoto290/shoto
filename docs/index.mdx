shoto is a Claude Code marketplace hosting plugins that help you build Claude Code itself: skills, sub-agents, hooks, and plugins — all authored through meta-tooling rather than hand-crafted files.

## Plugins

<CardGroup cols={2}>

  </Card>
  <Card title="git" href="/plugins/git/overview">

  </Card>
  <Card title="docs" href="/plugins/docs/overview">
