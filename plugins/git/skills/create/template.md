# PR body template

Fill in this skeleton with 1 to 4 bullets, then hand the result to `gh pr create --body`.

```
## Summary

- <imperative sentence describing a user-visible change>
- <imperative sentence describing another user-visible change>
```

For the wording rules, sample outputs, the opt-in `## Test plan` section, and what to avoid, see [reference/pr-body-rules.md](./reference/pr-body-rules.md).
