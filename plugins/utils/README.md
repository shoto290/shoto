# utils

Shared utility skills used by other `shoto` plugins.

## Install

```bash
# inside Claude Code
/plugin marketplace add shoto290/shoto
/plugin install utils@shoto
```

## What's inside

### Skills (`/utils:<name>`)

| Skill | Purpose |
| :--- | :--- |
| `/utils:explore` | Explore a codebase area with selectable profiles (`survey`, `deep-dive`, `targeted`) and return a compact, location-anchored report. |
